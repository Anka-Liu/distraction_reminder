# 快速开始指南

## 第一步：生成图标文件

1. 在浏览器中打开项目根目录的 `generate-icons.html` 文件
2. 点击三个下载按钮，下载 icon16.png、icon48.png 和 icon128.png
3. 将下载的三个图标文件放到 `public` 目录中

## 第二步：构建扩展

打开终端，在项目目录运行：

```bash
npm run build
```

构建完成后，`dist` 目录就是完整的扩展包。

## 第三步：安装到浏览器

### Chrome 浏览器

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目的 `dist` 目录

### Edge 浏览器

1. 打开 Edge，访问 `edge://extensions/`
2. 开启左下角的"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择项目的 `dist` 目录

## 第四步：配置摸鱼网站

1. 在扩展列表中找到"摸鱼时间控制器"
2. 点击"详细信息"或右键选择"选项"
3. 在设置页面添加你的摸鱼网站
   - 例如：名称 "哔哩哔哩"，URL "bilibili.com"，时长 600 秒（10分钟）

## 第五步：开始使用

1. 访问你添加的摸鱼网站（如 bilibili.com）
2. 页面右下角会显示倒计时
3. 倒计时结束后会自动跳转到 Google（或你设置的其他网站）

## 提示

- 如果需要更多时间，在设置页面点击"延时"按钮
- 可以随时在设置页面查看总摸鱼时间
- 修改设置后需要刷新正在访问的网站才能生效
